How it started

I started this project because I wanted to develop my programming skills, I chose python as the main language in the program so everytime you use one of the main features of the program, the code that is running is python, for the style (ex: the way that the games are displayed) I used bootstrap's CSS which is a library of CSS that allows you to easily add style to the content displayed on HTML. I used an html templating engine (lodash) to generate the game cards and jquery to interact with the html.

The user interface

I wanted to have a simple but modern user interface so i decided to use eel, since It uses Google Chrome to display the content it's just a matter of creating a website running on localhost that executes python scripts that interact with your computer to find where are the saves located and make the respective backups if desired
