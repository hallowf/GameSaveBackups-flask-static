Program not working or behaving as expected

Since this program opens a Google Chrome window, the first thing you should do if something went wrong is to open google chrome, go to settings scroll down and open advanced, the first settings should be related to privacy and security, on there select clear browsing data and delete cached images and files(On top of the pop-up that opens there is a Time range i reccomend setting it to the time you've been using the program or just select "All time")
