Program not working or behaving as expected

Since this program opens a Google Chrome window, the first thing you should do if something went wrong is to open google chrome, go to settings scroll down and open advanced, the first settings should be related to privacy and security, on there select clear browsing data and delete cached images and files(On top of the pop-up that opens there is a Time range i reccomend setting it to the time you've been using the program or just select "All time")

How to run the program with console output

Starting the program with a console might help you understand what is happening when you run the program or click certain buttons, that happens because troughout the code are various logs to the console some are python prints so they will be displayed in your operating system's console and the other are javascript logs that show up in Google Chrome's console

To run the program with a console

  • On windows

    • Open Window's start menu, type cmd and press enter, that will display the console, now you need to navigate to your program folder simply click the folder to open it and on the top there should be it's location copy it and on the console type "dir" without quotes and paste it (you can use ctrl + shift + v to paste in the console)
